# EMenu Technology — Website

Marketing site for [emenutechnology.com](https://emenutechnology.com) — digital menus and smart ordering for restaurants.

Built with Next.js 15, React 19, and Tailwind CSS 4.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
```

## Deploy

Connect this repo to Vercel or Netlify — build command `npm run build`, no environment variables needed.
